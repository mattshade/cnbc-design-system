# Tool Logos

Place the following logo files in this directory:

- `antigravity.svg` - Google Antigravity logo (colorful A)
- `claude.svg` - Claude/Anthropic logo (orange/coral knot)
- `copilot.svg` - GitHub Copilot logo
- `codex.svg` - OpenAI Codex logo
- `cursor.svg` - Cursor logo

Logos should be:
- SVG format (preferred) or PNG with transparent background
- Square aspect ratio (will be displayed in rounded containers)
- Optimized for web (< 50KB)
